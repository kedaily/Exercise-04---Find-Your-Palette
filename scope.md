### Project Purpose
    The purpose of this project is to design and develop a simple personal introduction webpage utilizing basic Markdown and HTML languages. 

### Project Description
    The project consists of creating a single page personal introduction document. The page will include the following
        1. Title element in the head.
        2. A main heading (h1) with name.
        3. A subheading (h2) for my major and/or professional interests
        4. At least two paragraphs (p) describing professional goals for the coarse and career
        5. An unordered list (ul) with at least three list items (li) detailing hobbies and interests
        6. A hyperlink (a) to my GitHub profile.
    The final project will be able to be viewed on GitHub or other platforms that support viewing 

### Desired Results
    The project should be able to convey the following results
*   A fully functional personal introduction page written in HTML.
*   A clear understanding  of basic HTML syntax and structure
*   A document that can be shared online via GitHub.

##  Exclusions
    The webpage is not about creating a visually stunning design, it is to show the understanding of basic HTML concepts and providing a well-structure document to build on the knowledge for future assignments. 
    No multi-page websites
    No use of advanced web developement like CSS.
